<a>texto</a>

Es una etiqueta que nos ayuda a poder crear un enlace a una página web. El atributo principal de la etiqueta HTML es href, donde pondremos el enlace al que queremos conectar. Otro atributo muy usado es target, el cual nos sirve para indicar si el enlace se abrirá en una nueva ventana o en la misma.

Uso:

<a>texto</a>

Atributos:

href | target | rel | hreflang | media | type

Etiqueta de Cierre: Requerida.

Posicion: Dentro de la etiqueta <body>

Atributos

href

href este atributo se utiliza para hacer referencia a otro documento. Puede encontrarlo en etiquetas de enlace (link) y etiquetas de anclaje. Sin embargo, los valores de la URL pueden ser diferentes dependiendo de lo que esté apuntando.

Enlace a otro documento

<a href="index.html">inicio</a>

inicio

Enlace al mismo documento hacia un ancla.

<a href="#inicio">inicio</a>

inicio

target

Target Este atributo se añade en los enlaces en una página para que al momento de hacer clic sobre ellos éstos abran en una nueva pestaña o no en la misma el documento que es llamado. El enlace target blank se utiliza para evitar que el usuario salga de la página donde está.

Variaciones de uso
  • _blank
  • _parent
  • _self
  • _top

Enlace a otro documento en la misma pestaña

<a href="index.html" target="_self" >inicio</a>

inicio

Enlace a otro documento en una nueva pestaña

<a href="index.html" target="_blank" >inicio</a>

inicio

rel

rel es un atributo obligatorio que especifica la relación entre el documento subyacente y el documento vinculado. El valor "alternate" indica al motor de búsqueda que el documento externo contiene una versión alternativa de la web.

hreflang

hreflang es un atributo que describe el idioma en el que está escrito el documento vinculado y, opcionalmente, también puede identificar el país para el que es particularmente relevante. Como valores se utilizan los códigos regionales y de idioma de acuerdo con las normas ISO 639-1 y 3166-1.

<a href="index.html" target="_blank" rel="alternate" hreflang="es-VE">inicio</a>
